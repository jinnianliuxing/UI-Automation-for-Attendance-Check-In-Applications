# 🚀 Auto Check-in System (某安自动打卡系统)

一款基于 Python + Tkinter + UIAutomation 开发的 Windows 桌面自动化工具，旨在解决每日重复性小程序打卡任务。支持定时执行、微信通知及开机自启。

## ✨ 核心功能

- 🤖 **全自动打卡**：自动唤醒屏幕、解锁登录、启动微信小程序并完成打卡流程。
- ⏰ **智能定时任务**：支持自定义打卡时间，内置防睡眠机制（允许息屏但阻止系统休眠）。
- 📱 **微信状态通知**：打卡完成后自动发送结果至“文件传输助手”，并可选择自动关闭微信。
- 💻 **开机自启支持**：一键配置 Windows 计划任务，实现登录后自动运行。
- 🛡️ **健壮性优化**：
  - 内置 DPI 适配，支持高分辨率屏幕。
  - 具备断网/卡顿重试机制及冲突弹窗自动处理。
  - 支持“错过打卡宽限期”补救功能。

## ⚠️ 使用前必读

- **微信环境**：请确保 PC 端微信已登录，且保持后台运行。
- **快捷方式**：需在桌面创建目标小程序的快捷方式（默认关键词：“中南林业科技大学学生工作部”，可在代码或配置中修改）。
- **锁屏设置**：建议关闭 Windows 登录密码或设置自动登录，否则无法在锁屏状态下自动进入桌面。
- **管理员权限**：首次使用“开机自启”或“防睡眠”功能时，请以管理员身份运行。

## 📥 安装与运行

1. 下载最新版本的 `.exe` 安装包。
2. 解压后右键点击主程序 -> **“以管理员身份运行”**。
3. 在界面中设置打卡时间，点击 **“定时打卡”** 即可。
